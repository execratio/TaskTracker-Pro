# TaskTracker-Pro

TaskTracker-Pro - это приложение для управления задачами и контроля личной продуктивности. Программа позволяет структурировать список дел, устанавливать приоритеты и отслеживать статус выполнения проектов.

## Функциональные возможности

* Создание, изменение и удаление задач.
* Категоризация задач по статусам: "К выполнению", "В работе", "Завершено".
* Установка уровней приоритетности.
* Хранение истории изменений в локальной базе данных.

## Системные требования

* Python версии 3.10 или выше.
* Установленный менеджер пакетов pip.
